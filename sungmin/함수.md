# 함수

## 함수 정의 방식

### 1. 함수 선언문
```js 
function add(x, y){
  return x + y;
}
```

### 2. 함수 표현식
```js 
const add = function(x, y){
  return x + y
}
```

### 3. Function 생성자 함수
```js 
const add = new Function('x', 'y', 'return x + y');
```

### 4. 화살표 함수(ES6)
```js 
const add = (x, y) => x + y;
```


## 즉시 실행 함수(IIFE, Immediately Invoked Function Expression)
```ts
// 익명 즉시 실행 함수
(function (){
  const a = 3;
  const b = 5;
  return a * b;
}());

// 기명 즉시 실행 함수
(function foo(){
const a = 3;
const b = 5;
return a * b;
}());

foo(); // ReferenceError: foo is not defined
```
