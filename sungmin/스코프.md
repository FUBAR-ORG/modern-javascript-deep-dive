# 스코프
스코프는 식별자가 유효한 범위를 말한다.

## 렉시컬 스코프

#### 동적 스코프와 정적 스코프
- 동적 스코프(dynamic scope): **함수를 어디서 호출**했는지에 따라 상위 스코프를 결정
- 정적 스코프(lexical scope): **함수를 어디서 정의**했는지에 따라 상위 스코프를 결정

```ts
var x = 1;
function foo(){
  var x = 10;
  bar();
}

function bar(){
  console.log(x);
}

foo();
bar();
```

- 자바스크립트는 렉시컬 스코프를 따르므로 어디서 정의했는지에 따라 상위 스코프를 결정
- 함수의 상위 스코프는 정의가 실행될 때 정적으로 결정
